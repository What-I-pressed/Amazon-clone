This project is a learning-oriented clone of an online store (similar to Amazon) built with Spring Boot for the backend, React for the frontend, and PostgreSQL as the database. It implements core e-commerce features such as product catalog, shopping cart, checkout, authentication, and admin management.


**Tech Stack
**
  Backend: Java, Spring Boot (Spring Web, Spring Data JPA, Spring Security)
  
  Frontend: React (Vite), React Router, Axios
  
  Database: PostgreSQL
  
  Build/Run: Maven for backend, npm for frontend
